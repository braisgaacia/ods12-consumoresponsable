# ODS 12 - Producción y Consumo Responsables

Sitio web sobre el **Objetivo de Desarrollo Sostenible 12** de la Agenda 2030.

## Estructura
- `index.html`: introducción y metas del ODS 12.
- `secundaria.html`: datos destacables, noticias y vídeo.
- `style.css`: estilo visual mejorado para menú y contenedores.
- `assets/`: imágenes locales en formato SVG para asegurar enlaces correctos.

## Requisitos implementados
- Dos páginas (`index.html` y `secundaria.html`).
- Menú con 5 enlaces en cada página (4 internos + 1 enlace a la otra página).
- Cuatro contenedores por página con fondo, borde, padding y márgenes.
- Página index con:
  - imagen principal,
  - texto de metas con encabezados, párrafos y dos imágenes en medio,
  - lista con 3 enlaces para ampliar información,
  - infografía.
- Página secundaria con:
  - imagen principal,
  - texto de datos destacables con encabezados, párrafos y dos imágenes en medio,
  - lista con 3 enlaces a noticias,
  - vídeo relacionado con el ODS 12.
